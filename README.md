# .:dotfiles:.
My configuration files.

## Installation

    $ git clone https://github.com/themattman/dotfiles.git ~/.dotfiles
    $ cd ~/.dotfiles
    $ ./bootstrap.py

Provide `-h` to bootstrap.py to get the help menu for the installation script.

## Requirements
Python 2.7.x
(Untested on other versions)
